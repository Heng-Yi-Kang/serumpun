
## Features

- **App Router**: Utilizes the latest Next.js App Router for nested layouts and simplified routing.
- **React Server Components**: Optimized for performance by rendering components on the server.
- **Font Optimization**: Automatically optimizes and loads [Geist](https://vercel.com/font) via [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts).
- **Fast Refresh**: Instant feedback during development with hot module replacement.
- **Vercel Ready**: Seamlessly integrated for deployment on the Vercel platform.

## Getting Started

First, run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
